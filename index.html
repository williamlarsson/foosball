• Win: 1 point
  • Loss: -1 point
  • Goal: +/- 0.5 point
  • Rank bonus: The team with the lower average in rank ( position in table ) will get the rank difference as percentage added to the round score
  • Example:
    Team 1 has an average rank ( position 2 and 4 ) of 3
    Team 2 has an average rank ( position 10 and 12 ) of 11
    The bonus will be 80%( (11-3) * 10) and the result will get multiplied by 1.8
    So if the result was a win with 10-8 to Team 2, the score is calculated:
    Score = 1 + ( ( 10-8 ) * 0.5 ) = 2
    Rank bonus = 1 + ( ( 11 - 3 ) / 10 ) = 1.8
    Total score = Score * Rank bonus = 3.6
